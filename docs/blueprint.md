# **App Name**: Palpiteiro Pro

## Core Features:

- Live Match Feed: Real-time list of upcoming sports matches featuring team flags, names, and event details.
- Quick Prediction Input: Mobile-optimized score inputs designed for rapid betting entry with tactile feedback.
- Supabase Integration: Store user predictions securely in a Supabase database to track user performance and history.
- AI Insight Tool: Uses a generative AI tool to provide performance-based commentary and likelihood analysis for match outcomes to help users make better guesses.
- Personal Scorecard: A clean visual interface to review saved predictions and their eventual accuracy.
- Point Ranking System: Calculates points based on result accuracy and displays them on a global community leaderboard.

## Style Guidelines:

- Primary color: Electric Violet (#7163FF) to evoke high-energy competitive excitement. Dark scheme base (#0F0E1A) with desaturated deep-hued background for a premium broadcast feel.
- Accent color: Neon Cerulean (#3B82F6) for primary call-to-actions and confirmation states.
- Font pairing: 'Space Grotesk' for headlines and score digits to provide a technical, statistical aesthetic; 'Inter' for body text and navigation labels for maximum legibility on mobile screens.
- Sharp, linear icons with 2px stroke width to maintain the modern, technical sports data vibe.
- Compact, card-based mobile-first layout with high-contrast containers and oversized touch targets for rapid data entry.
- Micro-interactions on prediction submission, such as card-glow effects and spring-based transitions for 'Palpite Salvo' success messages.
